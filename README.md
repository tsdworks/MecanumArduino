# MecanumArduino
 Mecanum Robot Move Control based on Arduino.

# Wireless Control
 Based on APC220 Module.
 
 Send (0 Speed Angle) or (1 Speed_X Speed_Y Speed_Rotate) via Serial to move the vehicle.

 Send (2) to stop the moving.
